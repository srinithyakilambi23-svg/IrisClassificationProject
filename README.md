# Iris Flower Classification Project

## Overview
This project is a Machine Learning classification project that predicts the species of Iris flowers using flower measurements.

The model classifies flowers into:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

based on:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Algorithm
- K-Nearest Neighbors (KNN)

---

## Features of the Project
- Data preprocessing
- Data visualization
- Label encoding
- Model training and testing
- Accuracy evaluation
- Confusion matrix generation
- Species prediction

---

## Dataset
The project uses the Iris Flower Dataset containing different flower measurements and species labels.

---

## Project Workflow
1. Load Dataset
2. Clean and preprocess data
3. Visualize dataset
4. Split training and testing data
5. Train KNN model
6. Predict species
7. Evaluate model accuracy
8. Display confusion matrix

---

## Output
The program displays:
- First 5 rows of dataset
- Missing values
- Dataset summary
- Pairplot visualization
- Model accuracy
- Classification report
- Confusion matrix
- Predicted flower species

---

## Accuracy
The model achieves approximately 95% to 100% accuracy.
